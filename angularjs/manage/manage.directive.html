Increase the number of available Custom Dimensions

Creating a new custom dimension slots can take a long time depending on the size of your database. Therefore it is only possible to do this via a command which needs to be executed on the command line.

To create a new custom dimension slot execute the following command within your Piwik installation: 

./console customdimensions:add-custom-dimension --scope=action
// for scope "action" or
./console customdimensions:add-custom-dimension --scope=visit
// for scope "visit"

If you want to create multiple new custom dimensions at once simply append the number of dimensions that shall be created. For example to create 5 new custom dimensions in scope action execute the following command: 

./console customdimensions:add-custom-dimension --scope=action --count=5
